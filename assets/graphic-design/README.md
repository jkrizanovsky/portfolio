# Graphic Design Assets

This folder contains images for the graphic design portfolio page.

## File Naming Convention
- Use descriptive names: `placeholder1.jpg`, `placeholder2.jpg`, etc.
- Supported formats: `.jpg`, `.png`, `.webp`
- Recommended size: 1920x1080 or higher for best quality

## Current Setup
The graphic-design.html page is configured to display 12 pieces.
