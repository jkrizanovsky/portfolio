# Video Editing Assets

This folder contains thumbnail images for the video editing portfolio page.

## File Naming Convention
- Use descriptive names: `placeholder1.jpg`, `placeholder2.jpg`, etc.
- Supported formats: `.jpg`, `.png`, `.webp`
- Recommended size: 1920x1080 or higher for best quality

## Current Setup
The video-editing.html page is configured to display 5 pieces.
Each thumbnail links to a video URL (YouTube, Vimeo, etc.).
