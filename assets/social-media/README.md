# Social Media Assets

This folder contains images for the social media portfolio page.

## File Naming Convention
- Use descriptive names: `placeholder1.jpg`, `placeholder2.jpg`, etc.
- Supported formats: `.jpg`, `.png`, `.webp`
- Recommended size: 1080x1080 (square) or 1080x1350 (portrait) for social media content
